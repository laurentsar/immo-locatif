Principe

Aide à l'investissement locatif sur 3 marchés (Toulouse, Espagne, Marrakech), sur le modèle d'une appli de reco : note d'opportunité par quartier (rendement + tendance + risque), simulateur (cashflow, fiscalité par pays), actus marché avec sentiment, et prix réels DVF pour Toulouse.

Données

⚠️ Les prix/loyers par quartier sont indicatifs (à ajuster, éditables via ✏️) — il n'existe pas d'API officielle gratuite temps réel pour l'Espagne/Marrakech. Pour Toulouse, le bouton « Prix réels DVF » récupère les ventes réelles de la base officielle DVF (data.gouv.fr) et calcule le prix/m² médian par code postal.

Actus : Google News RSS par marché. Rien n'est un conseil en investissement.

🏘️ Annonces Idealista (Espagne)

Affiche de vraies annonces (prix, m², pièces, €/m²) pour les villes espagnoles, via l'API Idealista. Clé gratuite (quota ~100 req/mois) à demander sur developers.idealista.com (ils envoient un apikey + un secret).
